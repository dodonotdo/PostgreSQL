## Math Operators

|S.No.|Symbol| Details|
|-----|-------|--------|
|1.|`+`|Addition|
|2.|`-`|Subtraction between two values|
|3.|`*`|Multiplication|
|4.|`/`|Division|
|5.|`^`|Exponent|
|6.|`@`|Absolute value|
|7.|`%`|Remainder|


**Examples:**

```bash
# Divide
SELECT name, salary/age FROM demo;
```
```bash
# Addition
SELECT age + salary AS sum_salary FROM demo;
```
```bash
# subtraction
 SELECT salary - age FROM demo WHERE id=1;
 ```
